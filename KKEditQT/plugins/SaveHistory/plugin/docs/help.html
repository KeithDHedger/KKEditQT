Backs up current document before saving, adding date and time to the file name.
Right click on the document tab for the context menu and select "Open Document History" to use xdg-open to open the relevent folder in your local filemanager.
Right click on the document tab for the context menu and select "Clear Document History" to delete all the history for the document ( not the doc itself! ).
Will not create a backup if you use "Save As .." until you save the new doc.
Backups are saved in ~/.KKEditQT/SaveFilesHistory under the absolute path to the file.
Clear and Open have no effect if there is no history for this file.
